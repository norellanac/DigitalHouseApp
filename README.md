## How to run it (At least for Android devices)
- yarn
- yarn start
- yarn test

Requires [Node.js](https://nodejs.org/)  to run.

Install the dependencies and devDependencies and start the server.

```sh
cd <ProjectFolder>
yarn
yarn start | yarn test
yarn android | yarn ios
```
